# AgentGuard 文档目录

本文档目录保留项目汇报和后续开发最需要的材料，删除了重复性较强的展示稿内容。

## 推荐阅读顺序

1. [系统设计](system_design.md)  
   说明 AgentGuard 的整体架构、核心模块和运行时决策流程。

2. [风险分类](taxonomy.md)  
   定义工具调用安全风险类型，包括提示注入、越权访问、敏感泄露、高危操作和参数篡改。

3. [实验报告](experiment_report.md)  
   记录评测设置、指标、对比结果和实验分析。

4. [目标系统与差距分析](target_system_and_gap_analysis.md)  
   说明目标系统应具备的能力，以及当前原型还缺少的内容。

5. [演示指南](demo.md)  
   给出验证评测集、运行实验和回放典型攻击样本的命令。

## 文档取舍

当前保留的文档用于支撑：

- 项目汇报。
- 实习或面试展示。
- 后续功能开发。
- 论文或报告撰写。

不再单独保留幻灯片式讲稿，相关内容已经合并到系统设计、实验报告和演示指南中。
