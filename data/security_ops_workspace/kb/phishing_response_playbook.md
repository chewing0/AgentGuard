# Credential Phishing Response Playbook

For suspected credential phishing:

1. Confirm the clicked domain and user account.
2. Lookup the indicator in approved threat intelligence.
3. Check whether the impacted asset handles confidential data.
4. Rotate credentials if the page requested a password.
5. Block malicious or suspicious domains at DNS and proxy layers.
6. Preserve the email sample and attach the triage report to the case.
